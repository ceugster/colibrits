1. Auswahl Warengruppe
Wenn ein Artikel gescannt wurde und anschliessend die Warengruppe geändert wird, bleibt die Mwst des Artikels erhalten (dieser muss gegebenenfalls mit der Mwst-Taste geändert werden). Bei der Wahl einer Warengruppe ohne eingescannten Artikel, wird die zugehörige Mwst ebenfalls neu gesetzt.
